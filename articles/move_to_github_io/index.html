dwango.github.io を公開しました

Tue, Oct 31, 2017
Tweet

ドワンゴ 技術コミュニケーション室の塩谷（ @kwappa / kwappa@friends.nico ）です。

ドワンゴではエンジニアブログとして自社プロダクトである「ブロマガ」を使ってきました。しかし、もともとエンジニア向けに作ったものではないので、技術記事を書くには機能的に苦しい部分もありました。

もっと記事を増やしていきたいという思いもあり、エンジニアが記事を書きやすいプラットフォームに一念発起して引っ越すことにしました。

以前より、github.com/dwangoで公開しているOSSを紹介するためにGitHub Pagesを利用してdwango on GitHubというページを公開していたのですが、ここを活用してエンジニア向けのコンテンツを発信していこうと思っています。

サイトの構築にはHugoというGolang製のジェネレータを使っています。Markdownで書けること、ChromaというSyntax Highlighterを同梱していること、テーマが充実していることなどの理由で選定しました。

ドワンゴ社内で用いている技術の活用事例などテクニカルなトピックはもちろんのこと、ドワンゴの社内やそこで働いているエンジニアをもっと身近に感じてもらえるような記事も増やしていく予定です。

それでは、dwango on GitHubをよろしくお願いいたします。新着記事はRSS feedなどからチェックできます。また、記事の執筆に興味のあるエンジニアの方はこちらからご連絡ください。
